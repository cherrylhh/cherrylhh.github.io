Github + hexo搭建blog

发表于 2018-03-07

Hexo是一个快速、简洁且高效的博客框架。Hexo使用Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

参考Hexo中文文档，记录一下在mac环境下搭建博客的步骤。

安装前部署环境

  • Node.js
  • Git

Github Pages创建

首先登录你的github帐号，创建一个仓库，仓库名字必须按照 yourname.github.io 格式命名。
